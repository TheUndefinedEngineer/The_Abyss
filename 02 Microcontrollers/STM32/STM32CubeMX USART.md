#### 1. **Asynchronous**
- Most common (what you use for GPS, Bluetooth, debugging)
- Only TX + RX
- Uses baud rate (no clock pin)

#### 2. **Synchronous**
- Adds a **clock pin (CK)**
- Data is synchronized with clock

#### 3. **Single Wire (Half Duplex)**
- TX and RX share same line

#### 4. **LIN (Local Interconnect Network)**
- Automotive communication

#### 5. **IrDA**
- Infrared communication

#### 6. **Smartcard**
- ISO 7816 protocol
