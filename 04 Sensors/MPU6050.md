Each range changes the **sensitivity (LSB per g)**.

|Range|Sensitivity|
|---|---|
|±2g|16384 LSB/g|
|±4g|8192 LSB/g|
|±8g|4096 LSB/g|
|±16g|2048 LSB/g|
### How to configure the range

The accelerometer range is set in register:

ACCEL_CONFIG (0x1C)

Bits **4 and 3** control the range.

|Bits|Range|
|---|---|
|00|±2g|
|01|±4g|
|10|±8g|
|11|±16g|

# Configuration register

Gyro range is set in:

GYRO_CONFIG (0x1B)

Bits **4:3** control the range.

|Bits|Range|
|---|---|
|00|±250|
|01|±500|
|10|±1000|
|11|±2000|

|Range|Sensitivity|
|---|---|
|±250 °/s|131 LSB/°/s|
|±500 °/s|65.5 LSB/°/s|
|±1000 °/s|32.8 LSB/°/s|
|±2000 °/s|16.4 LSB/°/s|
